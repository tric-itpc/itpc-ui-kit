# ITPC UI Kit

UI-библиотека компании ТРИЦ

## Установка

```sh
// with npm
npm install itpc-ui-kit

// with yarn
yarn add itpc-ui-kit
```

## Использование

```typescript jsx
import React, { useState } from "react"
import { Button } from "itpc-ui-kit"

const MyComponent: React.FC = () => {
  const [isOpen, setIsOpen] = useState<boolean>(false)

  return <Button onPress={() => setIsOpen(!isOpen)}>Click!</Button>
}
```

## Стилизация и темная тема

Для подключения стилей, необходимо использовать файл `index.css`.
Внутри него содержится описание стилей для каждого компонента и темной темы.
Переключение темной темы осуществляется путем подключения провайдера и использования хука `useTheme`.

```typescript jsx
import { Checkbox, ConfigProvider, useTheme, Theme } from "itpc-ui-kit"

const App: React.FC = ({ children }) => {
  const { theme, toggleTheme } = useTheme()

  return (
    <ConfigProvider>
      <Checkbox checked={theme === Theme.DARK} onClick={toggleTheme} />

      {children}
    </ConfigProvider>
  )
}
```

Так же для включения темной темы, достаточно добавить класс `itpc-theme-dark` к элементу `body`.
